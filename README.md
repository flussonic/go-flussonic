# Flussonic Go SDK

A Go SDK for interacting with Flussonic Media Server, Flussonic Watcher, Flussonic Central and Flussonic Vision APIs.

## Overview

This SDK provides a simple and idiomatic Go interface for:

- Managing Flussonic Media Server instances
- Configuring streams and DVR
- Interacting with Flussonic Vision for video analytics
- Managing TLS/SSL certificates

## Installation

```bash
go get flussonic-go-sdk
```
